# Deprompt

**Deprompt** is a browser extension that helps users become more conscious of their AI usage by tracking, visualizing, and optionally limiting interactions with popular AI chat platforms.

---

## What Deprompt Does

Deprompt runs entirely in the browser and provides the following core capabilities:

- **Usage tracking**  
  Measures the time spent on supported AI platforms.

- **Limits & blockers**  
  Allows users to define:

  - Time limits
  - Fixed blocking windows
  - Manual blocks

- **Interruptive nudges**  
  Optional reminders when usage exceeds configured thresholds or becomes continuous.

---

## Supported Platforms

Deprompt currently supports a growing set of AI chat providers, including:

- OpenAI ChatGPT
- Anthropic Claude
- Google Gemini
- Microsoft Copilot
- Poe
- Perplexity
- Meta AI
- Mistral
- Qwen
- Grok
- and others

(See `manifest.v3.json` for the authoritative list.)

## Installation

To use Deprompt, please download the latest release from GitHub and load it as an extension in your browser of choice.

As of now, there is no official build published in browser extension stores.
